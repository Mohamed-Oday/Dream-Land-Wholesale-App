<objective>
## Goal
Add OpenStreetMap location picker to store creation/editing form, display store location on detail screen, and give drivers the ability to create stores in the field.

## Purpose
Drivers visiting a new store can register it directly with GPS coordinates. Owner/admin can see store locations on a map. Closes the deferred "store location picker on map" issue from Phase 1.

## Output
- Store form: embedded map with tap-to-set location
- Store detail: mini-map showing store position
- Driver shell: Stores tab for field store creation
</objective>

    Add optional `gpsLat` and `gpsLng` params:
    ```dart
    Future<Map<String, dynamic>> create({
      required String name,
      String address = '',
      String phone = '',
      String contactPerson = '',
      double? gpsLat,
      double? gpsLng,
    }) async {
      final data = <String, dynamic>{
        'business_id': _businessId,
        'name': name,
        'address': address,
        'phone': phone,
        'contact_person': contactPerson,
      };
      if (gpsLat != null && gpsLng != null) {
        data['gps_lat'] = gpsLat;
        data['gps_lng'] = gpsLng;
      }
      final result = await _client.from('stores').insert(data).select().single();
      return Map<String, dynamic>.from(result);
    }
    ```
